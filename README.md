# Deepfake Image Forensic Analyzer

A Streamlit-based image forensic analysis tool.

## Features

- User Authentication
- Detection History
- Error Level Analysis (ELA)
- FFT Frequency Analysis
- Multi Image Upload
- Download Reports

## Technologies

- Python
- Streamlit
- SQLite
- NumPy
- Pillow
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
